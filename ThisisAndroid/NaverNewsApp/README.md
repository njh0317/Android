# Naver News App

## Desciption
네이버 News 검색 API를 사용한 간단한 네이버 뉴스 검색 어플리케이션입니다.
검색 창에 입력한 키워드와 유사한 뉴스를 RecyclerView로 나타냅니다.

## API
[**Naver News API**](https://developers.naver.com/docs/serviceapi/search/news/news.md#%EB%89%B4%EC%8A%A4)

## Design Patter
+ MVVM

## Used Tool
+ RecyclerView
+ Coroutine
+ Retrofit
+ View Binding
