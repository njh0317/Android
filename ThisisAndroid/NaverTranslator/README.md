# Naver Translate App

## Desciption
네이버 papago API를 사용한 간단한 번역 어플리케이션입니다.
입력창에 입력한 문장을 실시간으로 번역해서 나타냅니다.
### 추가기능
+ 한국어 <-> 영어 번역 기능
## API
[**Naver Papago API**](https://developers.naver.com/docs/papago/papago-nmt-overview.md)

## Design Pattern
+ MVVM

## Used Tool
+ Coroutine
+ Retrofit
+ View Binding
